ELECTION DESIGN CANVAS

ระบบการเลือกตั้งที่โปร่งใส
และตรวจสอบได้

Trustless & Traceable Election System Framework

เปลี่ยนจาก "ความเชื่อใจ" เป็น "คณิตศาสตร์" ให้ประชาชนเป็นผู้ตรวจสอบ

1. คุณค่าหลัก (Value Propositions)

Trustless Verification

ไม่ต้องเชื่อใจ กกต. หรือคนนับคะแนน แต่เชื่อมั่นใน Mathematics (Hash) และหลักฐานดิจิทัลที่อยู่ในมือคุณเอง

Crowd-Sourced Fraud Detection

ประชาชนทุกคนคือผู้ตรวจสอบ (Auditor) หากพบความผิดปกติแม้แต่ใบเดียว สามารถนำไปสู่การร้องเรียนเพื่อเลือกตั้งใหม่ได้

Full Traceability

ตรวจสอบเส้นทางของคะแนนได้ตั้งแต่หย่อนบัตร ย้ายหีบ แกะกล่อง จนถึงการสแกนและแสดงผลบนหน้าจอ

2. ขั้นตอนการเลือกตั้ง (Key Activities)

1

User Identification

ยืนยันตัวตนด้วยบัตรประชาชน เพื่อตรวจสอบสิทธิ ณ หน่วยเลือกตั้ง

2

Receive Ballot

รับบัตรเลือกตั้งแบบกระดาษ (Physical Ballot) พิมพ์แบบ On-demand เฉพาะบุคคล

3

Secure Voting

กากบาท และหย่อนบัตร 2 ใบลงในตู้ที่มี 2 ช่อง (แยกสีชัดเจน) เพื่อป้องกันความผิดพลาด

4

Centralized Scanning

หีบบัตรเคลื่อนย้ายไปส่วนกลาง (กทม.) แกะกล่องและนำเข้าเครื่องนับคะแนน High-Speed Scanner & OCR

5

Real-time Verification

คะแนนขึ้นหน้าเว็บทันที ประชาชนตรวจสอบผ่านแอปทางรัฐ และสแกน QR Code บนเว็บเพื่อ Verify ความถูกต้องของ Hash

3. ทรัพยากรหลัก (Key Resources)

  • Ballot Printer Kiosk
    เครื่องพิมพ์บัตรที่มี Algorithm สร้าง Hash ผูกกับ User ID (ทำงานเบื้องหลัง) ป้องกันบัตรเขย่ง
  • High-Speed Scanner & Counter
    เครื่องนับคะแนนความเร็วสูง อ่านค่า OCR และบันทึกภาพดิจิทัลความละเอียดสูงพร้อมกัน
  • "Thang Rat" Super App
    โครงสร้างพื้นฐานดิจิทัลเชื่อมต่อฐานข้อมูลประชาชน ใช้สำหรับตรวจสอบสถานะการเลือกตั้งส่วนบุคคล
  • Transparency Server
    ระบบฐานข้อมูลกลางที่เปิดให้ประชาชน Query ข้อมูลภาพบัตรเลือกตั้งของตนเองได้ตลอดเวลา

Secure Infrastructure

4. ช่องทาง (Channels)

On-Site

หน่วยเลือกตั้งทั่วประเทศ สำหรับการยืนยันตัวตนและลงคะแนนแบบกระดาษ

Digital Platform

เว็บกลางแสดงบัตรเลือกตั้งทั้งหมด (Public) และแอปทางรัฐ (Private) สำหรับยืนยันความเป็นเจ้าของ

5. ความสัมพันธ์ (Customer Relationships)

Empowerment

คืนอำนาจให้ประชาชน "เป็นเจ้าของ" คะแนนเสียงอย่างแท้จริง

Direct Feedback Loop

หากสแกน QR แล้วข้อมูลบนเว็บไม่ตรงกับในมือถือ สามารถกด Report แจ้ง กกต. ผ่านแอปได้ทันที
