คุณค่าหลัก (Value Propositions)

ไม่ต้องเชื่อใจใคร (Trustless)

ไม่ต้องพึ่งพาความซื่อสัตย์ของเจ้าหน้าที่ กกต. แต่เชื่อมั่นในรหัสลับ (Hash) และหลักฐานในมือคุณเอง

ทุกคนคือผู้ตรวจสอบ (Auditor)

ประชาชนทุกคนช่วยกันจับโกงได้ หากพบความผิดปกติแม้แต่ใบเดียว สามารถนำไปสู่การเลือกตั้งใหม่ได้

ตรวจสอบเส้นทางได้ (Traceable)

เห็นเส้นทางของคะแนนตั้งแต่หย่อนบัตร จนถึงการสแกนและขึ้นโชว์บนหน้าจอ แบบ End-to-End

ทรัพยากรหลัก: ตู้พิมพ์บัตร (Kiosk)

แก้ปัญหา "บัตรเขย่ง"

ตู้ Kiosk จะพิมพ์บัตรเลือกตั้งออกมาก็ต่อเมื่อมีผู้มีสิทธิตัวจริงมายืนยันตัวตนเท่านั้น

  • On-demand: ไม่มีการพิมพ์ทิ้งไว้
  • Unique Hash: สร้างรหัสลับเฉพาะตัว
  • Real-time DB: เช็คสิทธิทันที

ทรัพยากรหลัก: สแกนเนอร์ความเร็วสูง

เปลี่ยนกระดาษเป็นดิจิทัล

High-speed scanners และระบบ OCR จะทำหน้าที่เปลี่ยนบัตรเลือกตั้งให้กลายเป็นหลักฐานดิจิทัลที่แก้ไขไม่ได้

  • Massive Throughput: สแกนได้เร็วสูง
  • OCR Technology: อ่านลายมืออัตโนมัติ
  • Instant Upload: ส่งข้อมูลขึ้นเว็บทันที

ช่องทาง: แอป "ทางรัฐ"

แอปพลิเคชันทำหน้าที่เป็น Secure Wallet สำหรับประชาชน

ใช้เก็บ "รหัส Hash ID" และ "ใบรับรองการเลือกตั้ง" เพื่อใช้ยืนยันว่าคุณไปใช้สิทธิแล้ว และใช้ตรวจสอบความถูกต้องของคะแนนภายหลัง

พลังแห่งการตรวจสอบ (Verification)

100%

ตรวจสอบด้วยตาตัวเอง

คุณไม่ต้องถามเจ้าหน้าที่ว่า "คะแนนถูกนับไหม?"

แต่คุณสามารถ "เห็นภาพบัตรที่คุณกากบาทด้วยมือตัวเอง" บนหน้าจอได้เลย

Empowerment: คืนอำนาจให้ประชาชนเป็นเจ้าของคะแนนเสียงอย่างแท้จริง

ช่วยกันจับโกง (Crowd-Sourced Fraud Detection)

Public Auditor Network

เมื่อข้อมูลบัตรทุกใบถูกเปิดเผย ประชาชนนับล้านคนจะช่วยกันเป็นตา (Auditor) ทำให้การโกงเป็นเรื่องที่เป็นไปไม่ได้

Real-Time Accountability

หากพบความผิดปกติแม้แต่จุดเดียว ระบบจะเปิดให้กด Report เพื่อนำไปสู่การตรวจสอบทันที
